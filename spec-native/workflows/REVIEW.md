# REVIEW.md

Workflow para revisar si una iniciativa esta realmente terminada.

## Checklist

- La spec sigue siendo correcta y refleja el alcance real.
- Todas las tareas tienen estado final consistente.
- La validacion cubre criterios de aceptacion relevantes.
- `../TRACEABILITY.md` permite navegar entre artefactos.
- `../DECISIONS.md` registra tradeoffs persistentes.
- No hay contradicciones entre documentos.
