# Workflows

Procedimientos repetibles para operar el framework.

## Objetivo

Reducir variacion entre sesiones y entre agentes cuando ejecutan una
misma clase de trabajo.

## Workflows base

- [`IMPLEMENTATION.md`](./IMPLEMENTATION.md):
  como pasar de spec a codigo validado.
- [`PLANNING.md`](./PLANNING.md):
  como descomponer una spec en tareas.
- [`REVIEW.md`](./REVIEW.md):
  como revisar si una iniciativa esta realmente cerrada.
